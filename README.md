# React Native Counter App

A simple mobile counter application built using **React Native** and **Expo**.

## Features
- **Increment:** Increases the count by 1.
- **Decrement:** Decreases the count by 1 (stops at 0).
- **Reset:** Sets the counter back to 0.
- **Milestones:** Displays a celebration message when the counter hits 10.

## Technical Concepts Used
- `useState()` for state management.
- `TouchableOpacity` for custom buttons.
- `StyleSheet` for layout and design.
- Conditional rendering for milestone messages.

## How to Run
1. Copy the code from `App.js`.
2. Paste it into a new [Expo Snack](https://snack.expo.dev).
3. View the app in the web player or on your mobile device via the Expo Go app.
